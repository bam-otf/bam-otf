# API References

## **[bamotf/server](./api/server)**

A server that exposes a REST API to create and manage payments.

## **[bamotf/node](./api/node)**

A Node.js client to interact with the _bamotf_ server.

## **[bamotf/react](./api/react)**

A React component library to easily integrate bitcoin payments in your React
app.

## **[bamotf/utils](./api/utils)**

A utility library to help with formatting and stuff like that.
